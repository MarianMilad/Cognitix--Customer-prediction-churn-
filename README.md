# Cognitix - Customer Churn Prediction

This project focuses on predicting customer churn using machine learning models.

## Overview
- Customer churn analysis and prediction
- Machine learning models for classification
- Data preprocessing and feature engineering

## Files
- `app.py` - Main application
- `Customer churn.ipynb` - Customer churn analysis notebook
- `Logistic regression.ipynb` - Logistic regression model notebook
- Models: XGBoost, Gradient Boosting, Logistic Regression
- Data files: Telecom customer churn datasets

## Usage
Run the application or notebooks to analyze and predict customer churn.
